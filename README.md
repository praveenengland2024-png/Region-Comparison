# Region-Based Sales Comparison Dashboard

## Objective
To compare sales and profit performance across two selected regions using an interactive Tableau dashboard.

## Features
- Dynamic region selection using parameters
- Sales and profit comparison
- Drill-down hierarchy (Region → State → City)
- Interactive dashboard

## Tools Used
- Tableau
- Parameters
- Calculated Fields
- Hierarchies

## Insights
- Helps identify top-performing regions
- Enables better business decision-making
